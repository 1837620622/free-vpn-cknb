# 免费 VPN 情报站

免费 VPN 情报站是一个公开情报聚合项目，自动整理免费节点、试用机场、机场索引、日报与评测内容。站点采用黑白双主题、全局 3D 科技背景和响应式布局，手机与桌面端都按可读性优先设计。

线上域名：

```text
https://free-vpn.chuankangkk.top
```

作者：**传康Kk**

## 项目亮点

| 模块 | 说明 |
| --- | --- |
| 多源聚合 | 自动抓取公开免费节点、试用机场、机场索引、评测与跑路名单 |
| 稳定去重 | 按名称和链接生成稳定 ID，跨来源合并重复条目 |
| 每日同步 | 后台每天同步一次，适合低频更新内容，避免无意义高频请求 |
| 省资源写入 | 先计算内容指纹，数据没变化时只记录运行状态，不重复写大快照 |
| 分类页面 | 首页、试用、免费、节点、机场库、日报、评测、数据源、关于 |
| 黑白主题 | 白色为日间模式、黑色为夜间模式，浅色主题下所有页面保持高对比 |
| 全局动效 | 使用 Three.js 制作全站 3D 数据核心背景，不遮挡正文内容 |
| 移动适配 | 手机端菜单、卡片、底部信息区均使用稳定网格，避免横向溢出 |

## 页面设计

| 页面 | 内容 |
| --- | --- |
| 首页 | 总览统计、重点分类入口、实时窗口、全局 3D 情报背景 |
| 试用 | 整理带试用周期、注册入口、优惠码或流量说明的机场 |
| 免费 | 展示免费套餐、公益节点和可公开访问的资源 |
| 节点 | 聚合公开订阅、协议标签、地区和更新时间 |
| 机场库 | 汇总试用、免费、索引与评测类条目 |
| 日报 | 按最新时间窗口展示近期新增与更新内容 |
| 评测 | 聚合公开评测、体验说明和参考文章 |
| 数据源 | 展示各来源状态、抓取数量和最近成功信息 |
| 关于 | 说明项目定位、数据策略、作者和声明 |

## 数据策略

| 项目 | 策略 |
| --- | --- |
| 同步频率 | 每天一次 |
| 写入判断 | 对条目核心字段计算内容指纹，未变化则跳过全量快照写入 |
| 活跃快照 | 线上只保留当前展示需要的活跃数据 |
| 运行状态 | 保留最近一次同步状态，便于确认任务是否正常 |
| 本地归档 | 过期内容进入 `data/archive.json`，不占用线上展示空间 |
| 失败处理 | 单个来源失败不会中断整体流程，页面仍读取最近可用数据 |

## 本地运行

安装依赖：

```bash
pnpm install
```

抓取本地数据：

```bash
pnpm scrape
```

启动开发服务：

```bash
pnpm dev
```

访问本地页面：

```text
http://localhost:3000
```

## 构建与发布

本地生产构建：

```bash
pnpm build
```

预览生产运行包：

```bash
pnpm preview
```

发布并绑定域名：

```bash
pnpm deploy
```

把本地 `data/vpns.json` 写入线上快照：

```bash
pnpm seed:remote
```

清理本项目构建缓存：

```bash
pnpm clean
```

## 常用命令
